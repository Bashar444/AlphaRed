# Dr. Bashar Al Agha - Professional Portfolio

A modern, responsive single-page portfolio website showcasing the professional journey of Dr. Bashar Al Agha - Founder & CEO of Omnium.

## About

This portfolio highlights:
- **Current Role**: Founder & CEO at Omnium - Professional Ecosystem Platform
- **Healthcare Expertise**: Dentist, Pharmacist, and Management Professional
- **Research**: 9+ research projects in healthcare, pharmaceutical sciences, and management
- **Education**: D.D.S, B.Pharm, and PGDM in Marketing

## Tech Stack

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS + shadcn/ui components
- **Animations**: Framer Motion
- **Build Tool**: Vite
- **Deployment**: Vercel

## Features

- ✨ Modern, responsive design with dark mode support
- 🎯 Single-page application with smooth scrolling navigation
- 📱 Mobile-responsive across all devices
- 🎨 Beautiful UI components from shadcn/ui
- ⚡ Fast performance with Vite build optimization
- 🌐 Integration with Omnium platform

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Sections

1. **Hero** - Introduction with professional title and Omnium affiliation

4. **Experience** - Professional roles including current CEO position at Omnium
5. **Research** - 9+ research projects across healthcare, pharmaceutical, and management domains
